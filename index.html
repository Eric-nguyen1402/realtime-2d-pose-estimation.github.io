Abstract

This project aims to develop a real-time 2D pose estimation system for colored objects using computer vision techniques. The system utilizes a color camera to capture video of a scene with different colored objects placed on a plane. The proposed methodology involves several key steps, including video acquisition, color segmentation, pose estimation using Principal Component Analysis (PCA), and conversion of pixel measurements to metric values using camera calibration parameters or homography. The software implementation is designed for optimal performance, targeting a processing speed of 10-20 frames per second. The results are visualized by overlaying the estimated poses as points and vectors on the input image. The project’s application areas include robotics, object tracking, industrial automation, and augmented reality.

These figures demonstrate the system’s ability to detect the pose and orientation of objects in real-time. In each scenario, three different colored objects (red, blue, and green) were placed on a surface. The orientation of the objects is depicted by the red line in the center, while the x and y coordinates represent the pose in millimeters. The figures provide a visual representation of the accurate pose estimation achieved by our system in real-time. From the figures, we see that the performance of the system in terms of accuracy is close to 100% as it can detect the pose of every object accurately.
